HTML link tag/a tag target Attribute

  • _blank    A target attribute with the value of “_blank” opens the linked document in a new window or tab.
  • _self    A target attribute with the value of “_self” opens the linked document in the same frame as it was clicked (this is the default and usually does not need to be specified).
  • _parent    A target attribute with the value of “_parent” opens the linked document in the parent frame.
  • _top    A target attribute with the value of “_top” opens the linked document in the full body of the window.
